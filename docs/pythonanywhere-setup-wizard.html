Pak Bank Discounts — PythonAnywhere Setup

Follow these steps in order. You already ran the initial clone/setup; this finishes the web app and scraper.

Step 1: Pull latest code (if you cloned before)

In PythonAnywhere Bash console:

cd ~/pak-bank-discounts-ai
git pull

Step 2: Install a2wsgi

cd ~/pak-bank-discounts-ai/backend
source venv/bin/activate
pip install a2wsgi

Step 3: Create web app

Option A — CLI (recommended):

  1. Go to Account → API Token and create a token
  2. Replace YOUR_TOKEN in the command below, then run:
cd ~/pak-bank-discounts-ai/backend
source venv/bin/activate
export PYTHONANYWHERE_API_TOKEN=YOUR_TOKEN
pa website create --domain ammarjamshed123.pythonanywhere.com --command '/home/ammarjamshed123/pak-bank-discounts-ai/backend/venv/bin/uvicorn --app-dir /home/ammarjamshed123/pak-bank-discounts-ai/backend --uds ${DOMAIN_SOCKET} app.main:app'

Option B — Manual (if CLI fails):

  1. Go to Web tab
  2. Add a new web app → Manual configuration → Python 3.10
  3. Source code: /home/ammarjamshed123/pak-bank-discounts-ai/backend
  4. WSGI file: /home/ammarjamshed123/pak-bank-discounts-ai/backend/wsgi.py
  5. Reload the web app

Step 4: Run scraper (load deals)

cd ~/pak-bank-discounts-ai/backend
source venv/bin/activate
python scripts/run_scrape.py

Wait 2–5 minutes. You should see: Scrape done: inserted X, expired Y

Step 5: Verify

  • Backend health
  • Deals API
  • Frontend
